📊 README — Secondary Order Booking Sales Analysis (January 2026)
📌 Project Overview

This project analyzes secondary order booking and delivery performance using real-world sales transaction data for January 2026. The dataset captures detailed operational, customer, employee, product, and delivery information, enabling comprehensive business performance analysis.

The objective of this project is to transform raw order booking data into actionable insights through structured data cleaning, aggregation, and dashboard reporting.

🎯 Business Objective

The analysis aims to:

Monitor order booking performance

Track delivery status and fulfillment efficiency

Evaluate sales employee productivity

Analyze customer and regional sales trends

Measure delivery value and order completion rates

Provide decision-ready insights for sales and operations teams
